## üéØ Key SEO Metrics to Monitor

### Google Search Console Metrics
- **Impressions**: How often your site appears in search results
- **Clicks**: Number of clicks from search results
- **CTR (Click-Through Rate)**: Percentage of impressions that result in clicks
- **Average Position**: Average ranking position in search results

### Core Web Vitals
- **LCP (Largest Contentful Paint)**: Should be under 2.5 seconds
- **FID (First Input Delay)**: Should be under 100 milliseconds
- **CLS (Cumulative Layout Shift)**: Should be under 0.1

### Technical SEO Health
- **Crawl Errors**: Should be 0 or minimal
- **Index Coverage**: Percentage of pages successfully indexed
- **Mobile Usability**: All pages should be mobile-friendly
- **HTTPS Status**: All pages should be served over HTTPS

## üìã SEO Emergency Checklist

### If Rankings Drop Suddenly
1. Check Google Search Console for manual actions
2. Review recent website changes or updates
3. Check for technical issues (server errors, broken links)
4. Verify robots.txt hasn't blocked important pages
5. Check if competitors have improved their SEO

### If Pages Aren't Being Indexed
1. Submit URLs manually in Google Search Console
2. Check robots.txt for blocking directives
3. Verify canonical URLs are correct
4. Ensure pages are linked from other pages on the site
5. Check for noindex meta tags
